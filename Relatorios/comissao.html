Relatório Comissões

Relatório de comissão

Feito um relatório com o intuito de mostrar, acompanhar e realizar download de dados relacionados às comissões pagas.

Componentes

Spinner, Numeral, axios, showError, downloadPdf, downloadExcel

API:

'/api/pedido/getcomissao'

#Request


    ##POST

    {
    "equipeVendas": "Number",
    "ano": "number/year",
    "mes": "number/month"
    }
